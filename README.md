# Emotion Detection using NLP and Machine Learning

## Overview
This project predicts human emotions from text using Natural Language Processing (NLP) and Machine Learning techniques.

## Features
- Text preprocessing
- Tokenization
- Stopword removal
- Emotion classification
- Model evaluation

## Technologies Used
- Python
- NLTK
- Scikit-Learn
- TensorFlow/Keras
- Pandas
- NumPy

## Dataset
- Kaggle Emotion Dataset
- Multiple emotion categories

## Emotions Detected
- Joy
- Sadness
- Anger
- Fear
- Love
- Surprise

## Model Performance
- Accuracy: 84%
- High precision across multiple emotion classes

## Workflow
1. Data Cleaning
2. Text Preprocessing
3. Feature Extraction
4. Model Training
5. Evaluation

## Results
The model successfully classified emotions with 84% accuracy.
